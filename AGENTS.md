This is a monorepo containing multiple sub-projects. List the root directory first, then navigate into the relevant sub-folder to find its detailed documentation and gain sufficient context before working on any task.

All cross-project specifications, conventions, and shared norms should be documented in this file.

## Language

This file must always be written in English.

## Package Managers

Use `uv` for Python environments. Use `pnpm` for Node.js/npm environments if needed.

## Commit Convention

Use [Conventional Commits](https://www.conventionalcommits.org/) for all commit messages. Format: `<type>(<optional scope>): <description>`. Common types: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`, `ci`, `perf`, `build`.
